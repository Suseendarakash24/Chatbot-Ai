# Job Market Intelligence Platform

## Overview

A Job Market Intelligence Platform that analyzes resumes and compares them with job market data stored in a MySQL database.

The system processes job market datasets through an ETL pipeline and provides:

* Resume Analysis
* Role Matching
* Skill Gap Detection
* Salary Insights
* Career Recommendations
* Relevant Job Opportunities

## Technology Stack

* Python
* Streamlit
* MySQL
* SQL
* Pandas
* PDFPlumber

## Features

* Resume Upload (PDF)
* Resume Information Extraction
* Skill Extraction
* Role Match Analysis
* Skill Gap Detection
* Salary Insights
* Relevant Job Opportunities
* Career Recommendations

## Workflow

Job Dataset
→ ETL Pipeline
→ MySQL Database
→ Streamlit Application
→ Resume Upload
→ Resume Analysis
→ Skill Extraction
→ Role Matching
→ Skill Gap Analysis
→ Career Recommendations
